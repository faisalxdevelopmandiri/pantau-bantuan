# Aset Brand Aplikasi

Letakkan file logo Anda di folder ini dengan nama berikut:

- `favicon.png` untuk ikon tab browser.
- `logo-mark.png` untuk logo utama kecil di header publik, sidebar, navbar mobile, dan kop cetak default.
- `logo-login.png` untuk logo di halaman login.

Catatan:

- Jika logo login sebenarnya sama dengan logo utama, salin file yang sama dan beri nama `logo-login.png`.
- Gunakan PNG transparan agar tampil bersih di latar putih.
- Ukuran praktis:
  - `favicon.png`: 32x32 atau 64x64 px.
  - `logo-mark.png`: minimal tinggi 256 px.
  - `logo-login.png`: minimal tinggi 256 px.

Selama file ini ada di folder `assets/brand`, `index.html` bisa dibuka lokal dengan dobel klik dan logo akan terbaca otomatis.
